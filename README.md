### 一般处理过程：
**灰度转换** ==>> **二值化处理** ==>> **图像识别**
<pre>二值化处理，选择合适的阈值:
    选取大概阈值范围，遍历区间，保存降噪后图片，根据降噪效果对比选择合理阈值
</pre>
